### üîß Main Settings:
1. **LockWithKeyOnly** ‚Äî Access to the vehicle is only possible with a key.

6. **EmptyKeyId** ‚Äî Identifier for an empty vehicle key.
7. **KeyIds** ‚Äî List of key IDs for vehicle access.
8. **KeyActionDistance** ‚Äî Distance at which a key can be used.
9. **ExcludedVehicles** ‚Äî List of ignored vehicles.
10. **Carjacking** ‚Äî Determines whether jacking up the vehicle is allowed.
11. **CarjackingOwnerOnly** ‚Äî Jacking up the vehicle is allowed only for the owner.
12. **RoadSurfaceTypes** ‚Äî Types of road surfaces where the vehicle can be used.
13. **OffRoadTireDamage** ‚Äî Whether tire damage is possible when driving off-road.
14. **OffRoadTireBreakSpeed** ‚Äî Speed at which tires break off-road.
15. **OffRoadCheckInterval** ‚Äî Time interval for checking vehicle condition when driving off-road.
